## To Get Start

Download GLAD: https://glad.dav1d.de/
Download GLFW: https://www.glfw.org/download.html
Sample source code:

g++ -Iinclude src/main.cpp src/glad.c -Llib -lglfw3dll -lopengl32 -lgdi32 -o myProgram
./myProgram.exe

## To Check the compiler version

g++ -dumpmachine
